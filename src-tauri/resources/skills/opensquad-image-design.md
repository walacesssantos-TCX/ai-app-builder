# OpenSquad Image Design

Esta skill encapsula localmente o artefato de design visual do OpenSquad.

## Fonte canonica interna

Carregue `references/original.md` antes de decidir:
- viewport e proporcao
- escala tipografica
- hierarquia visual
- batch consistency entre slides
- limites de legibilidade e clipping

## Papel no pipeline

Use esta skill sempre que houver:
- composicao visual de post
- geracao de HTML de slide
- definicao de ritmo, contraste e espacamento
- validacao de integridade vertical do canvas

## Ordem de uso

1. Ler o design system do cliente
2. Ler `references/original.md`
3. So depois compor a peca ou a sequencia de slides

## Restricao

Nao consulte `opensquad-master` nem caminhos externos. A referencia local em `references/original.md` e a unica fonte desta skill.
